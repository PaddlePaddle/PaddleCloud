PaddlePaddle Cloud

{% blocktrans %} PaddlePaddle是开源、易用的深度学习平台，为深度学习技术及应用国家工程实验室提供深度学习提供开放的资源

  • 便捷、高效的弹性计算资源：为深度学习研究者和从业者提供强大的、先进的弹性GPU资源，免去使用者承担昂贵的专业级GPU硬件设备成本和维护成本；
  • ServerLess：使用者几乎不用考虑如何编写执行分布式程序，直接提交训练程序，训练任务便可以扩展到多台服务器的多个GPU上分布式的执行；
  • 即写即运行：每个使用者都拥有自己独立的Jupiter Notebook环境并可以在Notebook中进行可视化的程序编写和运行即时查看运行效果；
  • 公开数据集：PaddlePaddle Cloud被授权提供多个公开数据集为研究者和从业人员使用，使用者可以使用这些公开数据快速构建深度学习模型并验证效果；
  • 先进的集群调度系统：PaddlePaddle Cloud使用Kubernetes作为集群任务的编排和调度器，可以将硬件资源利用率提升到80%，并且支持大量的使用者任务同时运行；
{% endblocktrans %}

PaddlePaddleCloud&国家工程实验室简介

为了解决我国人工智能基础支撑能力不足等问题，由国家发展和改革委员会批复， 百度牵头筹建“深度学习技术及应用国家工程实验室”，并于3月2日正式揭牌启动。
实验室由百度深度学习实验室主任林元庆、百度深度学习实验室杰出科学家徐伟、 清华大学张钹院士和北京航空航天大学李未院士组成的团队牵头。与清华大学， 北京航空航天大学，中国信息通信研究院，中国电子技术标准化研究院等共建单位一起， 推动我国深度学习技术及应用领域的产学研标用的全面发展。
“深度学习技术及应用国家工程实验室”将着重发力于深度学习技术、计算机视觉感知技术、 计算机听觉技术、生物特征识别技术、新型人机交互技术、标准化服务、深度学习知识产 权七大方向，建设“国内领先、世界一流”的深度学习技术及应用研究机构，从研究突破、 产业合作、技术成果转让、人才培养等方面提升我国人工智能领域整体竞争力。
同时，百度将向实验室开放其处于全球领先水平的深度学习必需三大资源——计算资源、 算法资源和大数据资源，以及产业最前沿的深度学习课题，吸引深度学习顶尖人才来参与 共同研究，并将借助实验室的人才培养机制和博士后站点等，为国家吸引和培养更多深度 学习人才。
在“深度学习技术及应用国家工程实验室”的运行发展过程中，百度将支持深度学习技术工 程平台建设，从软件、设备和人才等方面推动建立大规模深度学习工程平台。该平台基于 PaddlePaddle构建，同时引入了世界先进的多机分布式并行计算系统PaddlePaddleCloud， 为深度学习研究者和从业者提供强大的、先进的弹性计算资源。


便捷、高效的弹性计算资源：为深度学习研究者和从业者提供强大的、先进的弹性 GPU资源，免去使用者承担昂贵的专业级GPU硬件设备成本和维护成本；
ServerLess：使用者几乎不用考虑如何编写执行分布式程序，直接提交训练程序， 训练任务便可以扩展到多台服务器的多个GPU上分布式的执行；

公开数据集：PaddlePaddle Cloud被授权提供多个公开数据集为研究者和从业 人员使用，使用者可以使用这些公开数据快速构建深度学习模型并验证效果；

先进的集群调度系统：PaddlePaddle Cloud使用Kubernetes作为集群任务的 编排和调度器，可以将硬件资源利用率提升到80%，并且支持大量的使用者任务 同时运行；
开源和企业应用：PaddlePaddle 和 PaddlePaddle Cloud同样适用 Apache License 2.0作为开源软件协议，不论企业还是个人都可以免费获得 这些软件，并在企业自己的硬件之上构建深度学习平台。
